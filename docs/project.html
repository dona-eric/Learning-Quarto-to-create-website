📊 Hack2Hiere Tech DataScience 2020

🎯 Scoring de Crédit Intelligent

Exploration approfondie des données pour identifier les variables clés corrélées à l’octroi de crédit. Développement d’un modèle de scoring permettant de prédire la probabilité d’acceptation avec une précision élevée.

🛠 Stack Technique : Pandas, Scikit-learn, Matplotlib, Classification supervisée

😄 Système de Reconnaissance d’Émotions

🎭 Détection d’Émotions en Temps Réel

Application innovante capable de détecter les émotions humaines (joie, tristesse, colère, surprise, etc.) en temps réel via webcam. Utilisation de réseaux de neurones convolutionnels pour une reconnaissance précise.

🛠 Technologies : OpenCV, Deep Learning, CNN, Keras, Python

📰 VeritaAI - Détecteur de Fake News

🔍 Intelligence Artificielle contre la Désinformation

Système intelligent de détection automatique des fausses informations circulant en ligne. Pipeline complet incluant prétraitement des textes, vectorisation avancée, modélisation ML et suivi des performances via MLflow.

🛠 Stack IA : NLP, TF-IDF, SVM, Random Forest, MLflow, Streamlit

Code Source Démo Verita

✨ Fonctionnalités : - Gestion CRUD des articles - Suivi temps réel des ventes - Dashboard analytique - Interface mobile Flutter
